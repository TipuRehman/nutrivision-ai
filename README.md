# AI Nutrition Coach

AI Nutrition Coach is an AI-powered web application that analyzes food images and provides detailed nutritional insights. The application acts as a virtual nutrition coach that helps users understand the nutritional value of their food.

Users can upload an image of a food item, and the system will estimate its calories, nutrients, and health value using artificial intelligence.

---

## Project Overview

AI Nutrition Coach is designed to make food analysis simple and accessible. By using AI-based image analysis, the system evaluates food items and provides an estimated nutritional breakdown.

The goal of this project is to demonstrate how artificial intelligence can assist people in making better dietary and health decisions.

---

## Features

- AI-powered food image analysis  
- Upload or drag-and-drop food images  
- Detailed nutritional breakdown  
- Estimated calories  
- Protein, carbohydrates, and fats analysis  
- Iron, fiber, and vitamins estimation  
- Health rating of food (Healthy / Moderate / Unhealthy)  
- AI-generated nutrition advice  
- Suggested questions users can ask about food  
- Custom question input for users to ask about the uploaded food  
- Clean and modern user interface  
- Mobile responsive design  

---

## How It Works

1. The user uploads a food image.
2. The AI analyzes the image.
3. The system identifies the food item.
4. Nutritional values are estimated.
5. A detailed nutrition report is displayed.
6. The user can ask questions about the analyzed food.
